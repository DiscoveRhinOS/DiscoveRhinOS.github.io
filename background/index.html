Access to ROSbloX

ROSblox are open. Login to a ROSbloX via SSH.

Username: rock
Password: rock

Changing a ROSbloX network configuration

The ROSbloX network interfaces are configured in /etc/netplan/01-net.cfg. We refer to netplan.io for further instructions how to modify its network configuration.

Updating a ROSbloX

The ROSbloX can be updated by executiong docker compose pull in the the main repository in the home folder.
